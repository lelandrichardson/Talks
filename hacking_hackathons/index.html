Package Managers ARE YOUR FRIEND

Node.js -> npm (~25.5k)
$ npm install {package name}
Ruby -> Gems (~53.2k) 
$ gem install {package name}
.NET -> Nuget (~83.5k)
$ Install-Package {package name}
PHP -> Composer (~8.5k)
$ php composer.phar install
JavaScript -> Bower* (~1.2k)
$ bower install {package name}
